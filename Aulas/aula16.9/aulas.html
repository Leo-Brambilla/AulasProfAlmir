Introdução ao HTML

Um novo mundo de possibilidades criativas

HTML (Hyper Text Markup Language)é utilizado para estruturar uma página da web e seu conteúdo. Por exemplo, o conteúdo pode ser estruturado em parágrafos, em uma lista com marcadores ou usando imagens, videos e tabelas.

HTML - O básico

Este curso abordará os conceitos básicos de HTML para oferecer a você uma experiência prática e permitir que você explore sua criatividade na web.

Conhecimento é poder

O curso foi desenvolvido para iniciantes e ensinará como criar uma página da Web do zero, bem como algumas tags HTML que podem ser úteis ao criar seus próprios layouts.

Experiência prática com exercícios práticos

O curso mostrará como adicionar imagens, texto, listas, etc. às suas páginas da web e como usar tags HTML como ou você também poderá modificar o código usado neste artigo ou escrever suas próprias tags HTML para criar páginas da web com vários layouts.

Meta Tags

HTML5 é a quinta versão do padrão HTML para desenvolvimento web. Ele se baseia nos sucessos e na usabilidade das versões anteriores do HTML. O HTML5 inclui muitos recursos novos, como incorporação de vídeo, áudio e imagem, geolocalização, soquetes da web e muito mais. Os desenvolvedores usam HTML5 para criar sites, aplicativos, e-books, revistas online e muito mais. HTML5 traz o poder da Internet para dispositivos móveis, tornando essas plataformas fáceis de usar.
